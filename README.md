# Aprendiendo-sobre-ciberseguridad
 Auditoría de Red con Zenmap (Nmap GUI)

Este proyecto consiste en un ejercicio de reconocimiento de red utilizando **Zenmap**, la interfaz gráfica oficial de Nmap. El objetivo es identificar servicios, puertos abiertos y topología de red en un entorno autorizado.

## 🛠️ Herramientas y Entorno
* **Zenmap / Nmap**
* **Objetivo:** scanme.nmap.org (Servidor de pruebas autorizado)
* **Tipo de Escaneo:** Intense Scan
